
# user-service

A microservice that focuses on user authorization. It uses keycloak as identity and access management solution.

## Deployment

To deploy this project run from the project's directory

```bash
  docker compose up --build
```


## Authors

## Features

- Registration
- Login
- Logout
- Refresh token
